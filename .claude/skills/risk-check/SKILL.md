# Risk Check Skill

## 説明

収益化事例の危険性・規約違反・法規制リスク・誇大表現を確認するスキル。
`results/cases.csv` のリスク評価を詳細に再確認します。

## 使い方

```
/risk-check                    # cases.csv 全体のリスク事例を確認
/risk-check --high-only        # リスク「高」の事例のみ
/risk-check --case 3           # 特定のケースID を詳細チェック
```

## 実行内容

1. `results/cases.csv` を読み込む
2. リスク評価の詳細チェックを実行:
   - 誇大表現パターンの検出
   - 証拠の強さの再評価
   - 日本の法規制への抵触確認
   - 利用規約違反の可能性
   - 詐欺パターンの検出
3. リスクレポートを `reports/daily/YYYY-MM-DD-risk.md` に出力

## チェック項目

### 自動検出するパターン
- 「誰でも」「確実に」「絶対に」等の断言表現
- 月収100万円以上の主張
- 高額講座・情報商材への誘導
- 証拠なし or 証拠不明の高収益主張

### 手動確認が必要な項目
- 売上スクリーンショットの真偽
- SNS自動投稿の規約適合性
- APIの商用利用条件

## 出力例

```
⚠️  リスクレポート — 2026-05-02

高リスク事例: 1件
  → Case 5: AI月収100万講座 [スキップ推奨]
    - 誇大表現: 「誰でも確実に」
    - 証拠なし
    - 高額情報商材への誘導（198,000円）

中リスク事例: 2件
  → Case 2: SNS投稿自動生成 [規約確認が必要]
    - Twitter API利用規約確認が必要
  → Case 4: AI画像販売 [要注意]
    - Booth のAI画像販売規約変更リスク
```
