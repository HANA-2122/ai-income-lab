# ============================================================
# 収集のコツ：
# ✅ GitHub URL がある → 証拠強度「高」になりやすい
# ✅ Stars/Forks が多い → スコアが上がりやすい
# ✅ 具体的な手順がある → 再現性が高く評価される
# ❌ 「誰でも確実に稼げる」系 → 自動的に減点・赤フラグ
# ❌ Telegram/LINE への誘導 → 赤フラグ検出
# ❌ system prompt / leak / jailbreak 系 → 大幅減点
