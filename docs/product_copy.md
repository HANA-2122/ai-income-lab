## 中程度の説明文（BOOTH/Gumroad 商品説明・note 記事本文用）

```
毎日 X や GitHub に大量の AI 収益化ネタが流れていますが、
その大半は誇大表現・証拠なし・すでに陳腐化した手法です。

AI Income Lab は以下を自動化します：

1. GitHub と Hacker News から候補を自動収集
2. 「採用/保留/不採用」と採用タイプ（直接採用/構造採用）を判定
3. 「本質は何か・誰の悩みか・何に転用できるか」を抽出
4. 10ジャンル（MT5/XAUUSD/Polymarket/SEO/AI画像/ゲーム解説/教材 等）への転用アイデアを生成

--

【こんな人に向いています】
✅ AI 収益化ネタを毎日チェックしているが「で、何を試せばいい？」で止まっている人
✅ 「真似しよう」ではなく「本質を抽出したい」と思っている人
✅ Claude Code や n8n を使っていて、収益化の転用先を探している人
✅ X や情報商材に踊らされず、自分の強みと掛け合わせたい人

【向いていない人】
❌ 「AIで確実に稼ぐ方法」を探している人（そういうツールではありません）
❌ 自動投稿・自動売買を実装したい人（このツールは情報整理専用です）

--

【技術要件】
- Node.js v18 以上
- インターネット接続（収集時のみ）
- 追加パッケージ不要（npm install 不要）
```

## 長い説明文（note 有料記事・Zenn 記事用）

```
# AI Income Lab の使い方と設計思想

## なぜこのツールを作ったか

X や HN で毎日見かける「AI で月○万円」という投稿。
その大半は「証拠なし」「再現不可」「情報商材への誘導」です。

一方で、本当に価値ある事例（GitHubに公開された再現可能なツール、
実測データ付きの報告）は埋もれています。

AI Income Lab は、この「ノイズと本物を分ける」作業を自動化します。

## 何をするツールか

### Step 1: 収集
GitHub Search API と Hacker News Algolia API を使って、
キーワードに関連するリポジトリ・記事を自動収集します。
「Source Leak」「system prompt 流出」「jailbreak」系は自動で大幅減点されます。

### Step 2: 評価
input.txt に貼ったネタを以下の基準で評価します：
- 証拠強度（GitHub URL / スクリーンショット / 動画 / 決済証拠）
- リスクの低さ（12種類の赤フラグ検出）
- Claude Code 実装可能性
- 市場妥当性（13カテゴリへの分類）

### Step 3: 転用
評価された事例の本質を、以下の転用先ジャンルに自動展開します：
- 業務AIスキルOS
- MT5/XAUUSD 分析ツール
- Polymarket 観察ツール
- SEO サイト・コンテンツ
- AI素材カタログ
- Claude Code 初心者向け教材
...など10ジャンル

## 重要な注意事項

このツールは「情報整理・リサーチ補助」ツールです。
- 投資助言ではありません
- 収益を保証しません
- 自動投稿・自動売買は実装されていません
```

## 注意事項テンプレート（必ず商品説明に含める）

```
【重要な注意事項】

本ツールはリサーチ・アイデア整理のためのツールです。

・投資助言ではありません
  XAUUSD・MT5・Polymarket に関連するアイデアは学習・情報整理用途です。
  投資の最終判断は必ずご自身で行ってください。

・収益を保証しません
  評価結果やアイデアは「検証の価値がありそうか」の一次判断です。
  実際の収益はご自身の実行と市場環境に依存します。

・自動投稿・自動売買はしません
  コンテンツの下書き生成機能はありますが、投稿は人間が確認後に手動で行います。

・危険なコンテンツは収集しません
  リーク・system prompt流出・jailbreak 系コンテンツは自動的に大幅減点されます。

n8n 本体・Claude Code・GitHub は本ツールに含まれていません。
各サービスの利用規約に従ってご利用ください。
```
