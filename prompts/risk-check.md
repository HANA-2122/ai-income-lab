# リスクチェック プロンプト

事例のリスクを多角的に評価するためのプロンプトテンプレート。
`scam-detector` エージェントまたは Claude Code に渡して使用。

---

## 基本リスクチェックプロンプト

```
以下のAI自動化収益化事例についてリスク評価を行ってください。

[事例内容をここに貼り付け]

以下の観点でチェックし、それぞれ「問題なし / 要注意 / 高リスク」で評価してください:

1. 法規制リスク
   - 日本の景品表示法・特定商取引法に抵触しないか
   - 金融商品取引法（投資・予測市場関連）に抵触しないか
   - 著作権法（自動収集・再配布）に抵触しないか

2. 利用規約リスク
   - 使用ツールの利用規約に違反しないか
   - APIの利用条件（商用利用OK？自動化OK？）
   - SNSの自動投稿・スクレイピング禁止規定

3. 詐欺・誇大表現リスク
   - 収益主張に証拠があるか
   - 「誰でも」「確実に」等の誇大表現はないか
   - 情報商材・高額講座への誘導はないか
   - 売上と利益を混同していないか

4. 技術的リスク
   - 依存するAPIやサービスが停止するリスク
   - セキュリティリスク（APIキー漏洩等）
   - コスト暴走リスク（API料金の青天井）

5. 競合・市場リスク
   - 参入障壁が低すぎてすぐコモディティ化するか
   - プラットフォームに依存しすぎていないか

最終評価: リスクレベル（低/中/高）と理由を1文で述べてください。
```

## クイックチェックリスト

事例を見たときに30秒で判定するチェック項目:

### 即スキップ（高リスク）
- [ ] 月収100万円以上の主張 + 証拠なし
- [ ] 「確実に」「誰でも」「絶対に」3つ以上
- [ ] 高額講座（5万円以上）への誘導
- [ ] 匿名 + フォロワー100人未満 + アカウント1ヶ月未満
- [ ] 自動売買・自動投資のBot
- [ ] ログイン必須サイトの自動スクレイピング

### 要注意（中リスク・要調査）
- [ ] SNS自動投稿（規約確認が必要）
- [ ] Polymarket等の予測市場取引（観察のみならOK）
- [ ] 売上スクリーンショットのみ（コスト不明）
- [ ] アカウント6ヶ月未満・フォロワー1000人未満
- [ ] 特定プラットフォームに強く依存

### 安全（低リスク）
- [ ] GitHubにコード公開（MIT等）
- [ ] 詳細な手順・README あり
- [ ] 収益主張なし or 少額で現実的
- [ ] 複数の独立した証拠あり
- [ ] 長期運用している実績あり（1年以上）

## 売上 vs 利益チェック

収益主張を見たら必ず以下のコストを推定する:

| コスト項目 | 目安 |
|-----------|------|
| Claude/OpenAI API | $5〜$200/月（使用量による） |
| サーバー代 | $5〜$50/月 |
| ドメイン | 月100〜1000円 |
| 有料ツール | n8n $20/月、Make $9/月等 |
| 画像生成API | Midjourney $10〜$60/月 |
| 時間コスト | 時給2000円として月20時間=4万円 |

**純利益 = 売上 - 全コスト - 時間コスト**

例: 「月7万円」の場合
- API代: 3000円
- サーバー: 1000円  
- 時間: 月5時間 × 2000円 = 1万円
- **純利益: 7万円 - 1.4万円 ≈ 5.6万円**（現実的な数字）
