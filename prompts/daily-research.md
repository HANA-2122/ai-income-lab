# デイリーリサーチ プロンプト

このプロンプトをClaude Codeに渡して、収集した素材の深掘り分析を依頼します。

---

## 使い方

`data/raw/` に保存した素材ファイルと一緒に以下のプロンプトをClaude Codeに渡す:

```
以下のファイルに保存したAI自動化収益化の事例を分析してください。

分析の観点:
1. 収益主張の信憑性（証拠の強さを0〜3点で評価）
2. 再現可能性（同じことを自分が試せるか）
3. 初期コスト・ランニングコスト（売上ではなく純利益を推定）
4. リスク（規約違反・法規制・詐欺の可能性）
5. 自分が48時間以内に試せる最小実験案

結果をMarkdown形式で出力してください。
各事例を## Case N: [タイトル] の形式で区切り、
results/cases.csv に追加できるよう構造化された情報を含めてください。
```

---

## 深掘り分析プロンプト（スコア上位事例用）

```
以下の事例についてより詳細な分析をお願いします:

[cases.csvの1行をここに貼り付け]

追加で調べてほしいこと:
1. この自動化を実装するのに必要なスキル・ツール・コスト
2. 競合の多さ（同じことをしている人の数）
3. 収益のスケーラビリティ（上限はいくら？）
4. リスクの詳細（何がまずくなるか）
5. 最初の1週間でやるべき具体的なタスクリスト
```
