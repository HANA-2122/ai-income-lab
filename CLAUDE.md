# AI Income Lab — CLAUDE.md

## プロジェクト概要

**AI活用事例・Claude Codeデモ・収益化主張・需要シグナルから、
収益化に転用できる構造を発見し、小実験に変換する研究ツール。**

「稼いだ事例を真似する」のではなく、「なぜ人が反応したか・何が可能になったか」という
**能力パターンと需要構造**を抽出し、自分の強みのある市場に転用することを目的とする。

転用対象ジャンル：投資/MT5 / XAUUSD / Polymarket / AI画像販売 / SEOサイト /
ゲームルール解説サイト / Claude Code初心者向け教材 / 業務自動化テンプレート /
小型SaaS / YouTube・X発信コンテンツ

## 運用ルール

### やること ✅
- `data/raw/` に手動で保存した情報（.md / .txt / .json）を入れる
- `npm run daily` で分析・レポートを実行する
- `results/cases.csv` で事例を管理・追跡する
- スコアが高い事例のみ小さな実験として試す

### やらないこと ❌（禁止事項）

| 禁止行為 | 理由 |
|---------|------|
| X API・ログイン必須サイトの自動スクレイピング | 規約違反リスク・まだ実施段階でない |
| 自動投稿・自動売買・自動購入 | 制御不能なリスク |
| Polymarketの自動売買Bot | 観察・分析フェーズのみ |
| 情報商材の購入・転売 | 高コスト・詐欺リスク |
| 著作権のある記事の無断スクレイピング・再配布 | 法的リスク |
| 個人情報を含むデータの収集・保存 | プライバシーリスク |

## 評価基準

### 収益主張の証拠強度

| レベル | 条件 |
|-------|------|
| 強（3点） | GitHub公開 + スクリーンショット + Stripe/売上画面など複数独立証拠 |
| 中（2点） | スクリーンショット1枚 or コード公開のみ |
| 弱（1点） | 文章のみ、証拠なし |
| なし（0点） | 匿名・証拠不明 |

### 注意フラグ（疑うべき表現）
- 「誰でも」「確実に」「絶対に」「爆益」→ 誇大表現の可能性
- 月収100万円以上（個人・自動化）→ 要検証、コスト確認必須
- 匿名 + 証拠なし → 信頼性低
- 情報商材・講座販売 → 高リスク・煽り構造を確認

### 売上 vs 利益の区別（重要）
- 「月○○万円稼いだ」は**売上**の可能性が高い
- API費用・広告費・ツール代・時間コストを差し引いた**純利益**を推定すること
- 「売上」と「利益」を混同している主張は証拠強度を下げて評価

## ジャンルバイアスの制御

- XAUUSD/MT5/投資は有力ジャンルだが、プロジェクト全体の**唯一の主軸ではない**
- 転用先は10ジャンルあり、月100万円は複数収益源の合計で狙う
- 毎回XAUUSD系ばかりが上位に来る場合はスコアリングが偏っている可能性を疑う
- 総合ランキングとは別に、**ジャンル別ランキング**を出して偏りを可視化する
- 1ジャンルに依存せず、小さく試して当たりを伸ばすことを優先する

### アイデア優先度の判断順序
1. **シグナルの質**（pain_signal > capability_demo > revenue_claim > suspicious_claim）
2. **MVPの即実行可能性**（2時間以内に試せるか）
3. **リスクの低さ**（自動投稿・自動売買・投資助言を含まないか）
4. **収益導線の明確さ**（どこでどう収益化するかが具体的か）
5. **自分の強みとの相性**（上記4点のタイブレーカー。過剰加点しない）

## スコアリング（100点満点）

| 項目 | 配点 | 評価基準 |
|-----|------|---------|
| 需要 | 20点 | 市場規模・検索需要・再現事例数 |
| 収益化しやすさ | 20点 | 収益額・収益化までの時間 |
| 再現性 | 20点 | コード公開・手順明確度 |
| 自分の強みとの相性 | 20点 | Claude Code / JS / Python活用度 |
| リスクの低さ | 20点 | 法規制・規約・詐欺リスク |

## データ形式（data/raw/ に入れるファイル）

### Markdown形式（推奨）
```markdown
## Case 1: [タイトル]

- **投稿者**: @ハンドル名
- **URL**: https://...
- **日付**: YYYY-MM-DD
- **カテゴリ**: SEO/コンテンツ | SNS自動化 | Polymarket | AI画像販売 | 情報商材 | SaaS/ツール
- **使用ツール**: Claude Code, Python, ...
- **収益主張**: 月XX万円（アフィリエイト等）
- **証拠**: スクリーンショットあり / GitHubリポジトリ / 証拠なし
- **リスク**: 低 | 中 | 高

**投稿内容**: [一行サマリー]
```

### JSON形式
```json
[
  {
    "source": "twitter",
    "author": "@example",
    "date": "2026-04-20",
    "claim": "Claude Codeで...",
    "revenue_amount": "月7万円",
    "tools_used": "Claude Code, Python",
    "category": "SEO/コンテンツ"
  }
]
```
